# cmd

## cmd到图形化界面的故事

​	计算机刚开始的时候，基本没有个人使用，都是政府，军队使用，因为造价高，占地广耗电高。 这时候的电脑基本没有像我们今天使用的图形化界面，都是黑屏下采取cmd命令。

​	1974年，有一个叫罗伯茨的人生产出了一台价格不足400美元的计算机。这在当时是很轰动的大新闻，因为那时只有政府和军方才有能力使用计算机，计算机太贵了，普通人甚至见都没见过。这台电脑搭载了一个图形化界面。

![施乐](..\img\施乐.jpeg)

这个非常丑的界面，引起了业界两位大佬的相爱相杀。就是乔布斯和比尔盖茨。

​	乔布斯看到了图形界面的将来，推出全球首款将图形用户界面和鼠标交互的小型我电脑——“Apple Lisa”。

![Apple Lisa](..\img\Apple Lisa.jpeg)

 	作为乔布斯平生的敌手，比尔盖茨宣布微软即将推出 Windows 1.0。

在苹果总部办公室，乔布斯当面怒斥比尔盖茨，“你在盗用我们的器械!”

比尔盖茨平地步说出了他平生中最经典台词之一，“我们都有个充裕的邻居，名叫施乐。有一天我闯进了他家，妄图偷走他的电脑，却发明你已经疾足先得。”

微软Windows 1.0早期的版本质彩艳丽。

![Windows 1.0](..\img\Windows 1.0.jpeg)

因为图形化界面直观易用，所以他的出现，让越来越多的人方便快捷的熟悉计算机并使用计算机。

但是图形化界面需要消耗更多的内存，运行的速度没有命令方式快。所以大家在电视上看到一些顶级的黑客，他们一般都是使用命令行的方式操作计算机。

## cmd命令

在windows当中也集成了cmd命令行方式，我们了解一下命令即可。

切换盘符

​	盘符名称 + 冒号

查看当前文件夹的内容

​	dir

进入单级文件夹

​	cd + 文件夹名称

回退单级文件夹

​	cd..

进入多级文件夹

​	cd + 文件夹名称1/文件夹名称2/...

直接退到盘符

​	cd\

清屏

​	cls

退出

​	exit

打开文件

​	文件夹+后缀名

删除层级关系很深的文件夹

​	rd 文件夹 /s
